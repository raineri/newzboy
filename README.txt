INSTALLATION

1. Make sure you install python 2.3 or higher:
        http://www.python.org/download/

2. Optionally, install the following modules if you would like:

    psyco - recommended, greatly speeds up file decoding
        http://psyco.sourceforge.net/download.html

    wxPython - optional, if you want a gui for file transfers
        http://www.wxpython.org/download.php
